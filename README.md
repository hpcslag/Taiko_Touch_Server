HOW TO COMPILER?
====
###Worning
Please don't use DevC++ or MinGW Compiler, this project use Visual Studio Library.
#####Visual Studio 2013
1. Open Visual Studio 2013 and new Project, Copy Taiko_TCP_Server.cpp to your project path.
2. Debug and run your project.
